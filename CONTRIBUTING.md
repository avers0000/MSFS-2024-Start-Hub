# Contributing

Thank you for your interest in this project.

## Pull Requests
External pull requests are not accepted.  
All development and code changes are performed exclusively by the project author.

## Issues
If you want to suggest an improvement or report a problem, please open an Issue.  
Discussion is welcome, but all decisions regarding changes are made by the author.

## Forks
You may create forks of this project under the terms of the MIT License.  
However, changes from forks will not be merged into the main repository.
